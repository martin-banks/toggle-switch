Toggle switch

Experiment with drag to create a toggle switch component that can be implemented from a factory function

Sample config code

			
const toggleOptions = {
	container: '.tester',
	callback: {
		on: ()=> console.log('Button on'),
		off: ()=> console.log('Button off')
	},
	color: {
		off: false,
		on: false
	},
	size: 'jumbo' // 'small', 'regular', 'large', 'jumbo'
}
